# Ранжирование смартфонов 

Этот проект реализует систему персонализированного ранжирования смартфонов на основе модели машинного обучения (LightGBM Ranker) и пользовательских предпочтений. Приложение позволяет загружать данные о смартфонах, настраивать веса критериев и получать топ-рейтинг с визуализацией.

---

## Основные возможности

- **Интеллектуальное ранжирование** с использованием обученной модели LambdaMART
- **Гибкая настройка весов**: цена, рейтинг, камера, батарея, RAM, новизна 
- **Фильтрация** по:
  - Брендам
  - Операционной системе
  - Процессору
  - Поддержке 5G
  - Цене, рейтингу, RAM, ёмкости батареи
- **Интерактивные визуализации**:
  - ТОП-10 смартфонов (горизонтальный bar chart)
  - Сравнение ML-оценки и пользовательской оценки (scatter plot)
- **Экспорт результатов** в CSV

## Как запустить
1) Установите зависимости:
```bash
pip install -r requirements.txt
```
2) Обучите модель(однократно):
```bash
python leaning_model.py
```
→ Создаст файл smartphone_ranker.txt

3) Запустите веб-приложение
```bash
streamlit run app.py
```

4) Откройте в браузере: http://localhost:8501

## Как работает ранжирование

Пользовательская оценка (User Score) рассчитывается по формуле:

`User Score = 0.25·рейтинг + 0.15·камера + 0.20·батарея + 0.15·RAM + 0.20·(1 − цена) + 0.05·новизна`


Веса можно настроить в интерфейсе.

ML-оценка (ML Score) — прогноз обученной модели LightGBM Ranker.

Финальный скор:

`Final Score = 0.6 · User Score + 0.4 · ML Score`

Устройства сортируются по убыванию `Final Score`
